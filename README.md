# datanode

#### 介绍
双向链表和二叉树的实现

#### 功能特点
    双向链表
        createByList():
            可根据已有列表快速创建双向链表
        create():
            可快速创建特定长度的双向链表（默认节点值为None）
        tolist():
            可将双向链表转换为列表
        delete():
            删除当前节点并将上一个节点与下一个节点相连
        可使用加号（+）将两个双向链表连接
        可使用减号（-）从一个双向链表中删除两链表共同节点
    二叉树
        createByList():
            可根据已有列表快速创建二叉树
        create():
            可快速创建特定的二叉树（默认节点值为None）
        tolist():
            可将二叉树转换为列表
        nodes() / len():
            返回子节点数量
        delNode():
            删除子节点
